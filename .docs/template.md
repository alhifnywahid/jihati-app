# **Kebijakan Privasi (Privacy Policy)**

Di Jihati - Amaliyah Santri RUA, kami sangat berkomitmen untuk melindungi privasi Anda dan menjaga standar tertinggi keamanan data sesuai dengan Undang-Undang Perlindungan Data Pribadi (UU PDP) No. 27 Tahun 2022. Kami memahami bahwa kepercayaan Anda adalah fundamental, terutama untuk aplikasi yang menyediakan konten keagamaan. Kebijakan Privasi ini menguraikan bagaimana kami mengumpulkan, menggunakan, memproses, dan melindungi data Anda ketika Anda menggunakan aplikasi Jihati. Dengan menggunakan aplikasi Jihati, Anda mengakui bahwa Anda telah membaca dan memahami kebijakan ini serta menyetujui praktik data kami sebagaimana dijelaskan di sini.

## **1. Data yang Kami Kumpulkan**

Sesuai dengan Pasal 4 UU PDP, kami menerapkan prinsip minimisasi data. Aplikasi Jihati dirancang dengan **privacy-by-design** dan **TIDAK mengumpulkan data pribadi Anda**.

**Data yang TIDAK Kami Kumpulkan:**

- Nama pengguna atau identitas pribadi
- Alamat email atau nomor telepon
- Informasi akun (aplikasi tidak memiliki sistem akun/login)
- Lokasi geografis
- Kontak dari perangkat Anda
- Foto, video, atau file multimedia
- Informasi keuangan atau pembayaran
- Data biometrik
- Alamat IP atau informasi jaringan

**Data Lokal yang Tersimpan di Perangkat Anda:**

Aplikasi hanya menyimpan data berikut **secara lokal di perangkat Anda** untuk meningkatkan pengalaman pengguna:

- **Riwayat Bacaan:** Catatan surah atau bacaan yang telah Anda buka (maksimal 50 bacaan terakhir)
- **Bookmark/Tersimpan:** Daftar surah atau bacaan yang Anda tandai sebagai favorit
- **Preferensi Aplikasi:** Pengaturan tema (terang/gelap) dan ukuran font Arab

**Penting:** Data-data tersebut **TIDAK** dikirim ke server kami atau pihak manapun. Semua data tersimpan dan tetap berada di perangkat Anda. Data akan terhapus secara otomatis jika Anda menghapus aplikasi.

## **2. Dasar Hukum dan Tujuan Pemrosesan Data**

Sesuai dengan Pasal 20 UU PDP, kami memproses data lokal di perangkat Anda berdasarkan:

- **Kepentingan Sah:** Untuk menyediakan fitur aplikasi yang lebih baik dan personal
- **Persetujuan Implisit:** Ketika Anda menggunakan fitur penyimpanan seperti bookmark dan riwayat

Kami menggunakan data lokal tersebut untuk:

- Menampilkan riwayat bacaan Anda sehingga mudah melanjutkan bacaan terakhir
- Menyimpan bacaan favorit yang telah Anda tandai
- Mengingat preferensi tema dan ukuran font yang Anda pilih
- Meningkatkan pengalaman pengguna tanpa mengorbankan privasi

**Kami TIDAK menggunakan data Anda untuk:**

- Pemasaran atau iklan
- Profiling atau analisis perilaku pengguna
- Berbagi dengan pihak ketiga
- Menghasilkan pendapatan

## **3. Perlindungan dan Keamanan Data**

Sesuai dengan Pasal 58 UU PDP, kami menerapkan langkah-langkah keamanan untuk melindungi data Anda:

- **Penyimpanan Lokal:** Semua data disimpan secara lokal di perangkat Anda menggunakan mekanisme penyimpanan aplikasi yang aman
- **Tidak Ada Transmisi Data:** Data Anda tidak pernah dikirim melalui internet ke server kami atau pihak manapun
- **Enkripsi Perangkat:** Data Anda dilindungi oleh enkripsi perangkat bawaan sistem operasi Android/iOS
- **Tidak Ada Server:** Kami tidak memiliki server yang menyimpan data pengguna, sehingga risiko kebocoran data dari server tidak ada
- **Kontrol Penuh:** Anda memiliki kontrol penuh atas data Anda dan dapat menghapusnya kapan saja

Karena data tersimpan lokal di perangkat Anda, keamanan data juga bergantung pada:

- Keamanan perangkat Anda (password, PIN, biometrik, enkripsi perangkat)
- Pembaruan sistem operasi perangkat Anda
- Tidak memberikan akses perangkat kepada pihak yang tidak berpercayaan

Kami tidak dapat bertanggung jawab atas keamanan perangkat Anda, namun kami mendorong Anda untuk selalu menjaga keamanan perangkat dengan baik.

## **4. Pembagian Data dengan Pihak Ketiga**

Sesuai dengan Pasal 26 UU PDP, kami dengan tegas menyatakan:

**Kami TIDAK membagikan, menjual, menyewakan, atau mentransfer data pribadi Anda kepada pihak ketiga manapun.**

Alasannya sederhana: kami tidak mengumpulkan data pribadi Anda sama sekali.

**Pihak Ketiga yang TIDAK Kami Gunakan:**

- Layanan analitik (seperti Google Analytics, Firebase Analytics)
- Platform iklan (seperti AdMob, Facebook Ads)
- Penyedia cloud untuk penyimpanan data pengguna
- Layanan tracking atau profiling
- Social media tracking

**Pengecualian - Fitur Berbagi:**

Aplikasi ini memiliki fitur berbagi konten (share button) yang memungkinkan Anda membagikan teks bacaan ke aplikasi lain melalui sistem berbagi Android/iOS. Ketika Anda menggunakan fitur ini:

- Anda secara aktif dan sadar memilih untuk membagikan konten tertentu
- Data dikirim langsung ke aplikasi yang Anda pilih (WhatsApp, Telegram, dll)
- Kami tidak mengontrol atau menyimpan data yang Anda bagikan
- Konten yang dibagikan tunduk pada kebijakan privasi aplikasi penerima

**Pengecualian - Link Eksternal:**

Aplikasi ini berisi link ke situs web eksternal untuk tujuan tertentu:

- **Formulir Masukan:** https://tally.so/r/wv6GeQ (untuk feedback pengguna)
- **Media Sosial:** Link ke akun media sosial PW FORMASI RUA Surabaya 

Ketika Anda mengklik link ini, Anda akan diarahkan ke situs eksternal yang memiliki kebijakan privasi mereka sendiri. Kami tidak bertanggung jawab atas praktik privasi situs-situs tersebut.

## **5. Hak Subjek Data Anda**

Sesuai dengan Pasal 5-11 UU PDP, Anda memiliki hak-hak penuh atas data Anda:

**Hak Akses:**

- Melihat semua data yang tersimpan melalui menu **Riwayat** (untuk riwayat bacaan)
- Melihat semua bookmark melalui menu **Tersimpan**
- Melihat preferensi Anda melalui menu **Pengaturan**

**Hak Koreksi:**

- Karena data yang tersimpan adalah preferensi dan riwayat, Anda dapat mengubah pengaturan kapan saja melalui menu Pengaturan

**Hak Penghapusan:**

- Menghapus riwayat bacaan secara individual atau keseluruhan melalui menu Riwayat
- Menghapus bookmark secara individual melalui menu Tersimpan
- Menghapus semua data dengan menghapus aplikasi dari perangkat Anda

**Hak Portabilitas:**

- Data tersimpan dalam format lokal di perangkat Anda
- Karena sifat data yang minimal (hanya preferensi dan riwayat), portabilitas data tidak relevan untuk aplikasi ini

**Hak Penarikan Persetujuan:**

- Anda dapat berhenti menggunakan aplikasi kapan saja
- Menghapus aplikasi akan menghapus semua data yang tersimpan

**Hak Keberatan:**

- Karena kami tidak mengumpulkan atau memproses data pribadi Anda untuk tujuan pemasaran atau profiling, hak keberatan tidak berlaku

Semua hak ini dapat Anda jalankan sendiri tanpa perlu menghubungi kami, karena Anda memiliki kontrol penuh atas data di perangkat Anda.

## **6. Penyimpanan dan Retensi Data**

Sesuai dengan prinsip pembatasan penyimpanan dalam UU PDP:

**Penghapusan Data:**

- Tidak ada cara untuk memulihkan data setelah aplikasi dihapus (kecuali jika Anda menggunakan backup perangkat dari sistem operasi)
